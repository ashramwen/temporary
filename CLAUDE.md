## Repository Overview

This is a media asset repository for the **HY-5085** project. It contains image files (PNG, JPG, GIF) organized in the `hy-5085/` directory. There is no application code, build system, or test suite.

## Asset Organization

All assets live in `hy-5085/` and follow these naming conventions:

- `P<n>_*.png/jpg` — numbered page/section assets (P1, P2, ... P14)
- `*_pc.*` — high-resolution/desktop variants of an asset
- `*_BG.*` — background images
- `*.gif` — animated assets
